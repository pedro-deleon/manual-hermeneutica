Paso 2 Formulas de expresión

Meta: Que el alumno note las diferentes formas o formularios que se usan en el texto bíblico y cómo influyen en la interpretación.

Aparte del estilo literario, el autor puede estar usando una formula. Para entender su escritura tenemos que descubrir la formula, haciendo preguntas propias a las intenciones del autor.

Hay formulas para cartas, pactos, poesía, sermones, profecías, etc

Carta o epístolas

  1. Salutación
  2. Acción de gracias
  3. Cuerpo
  4. Exhortaciones morales
  5. Salutación y Bendición
Preguntas para estudio bíblico
  1. ¿Existe en nuestro texto una fórmula que está siguiendo el autor?
  2. ¿Cómo nos ayuda entender el texto, el tener conocimiento de esta fórmula?
